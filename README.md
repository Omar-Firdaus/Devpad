# Devpad
DevPad is a custom macropad built entirely from scratch, including PCB design, firmware, and hardware layout. It features a 3×3 key matrix, dual rotary encoders, a function button, and an integrated display. Designed for efficient dev workflows with fully custom electronics and hand-built components.
